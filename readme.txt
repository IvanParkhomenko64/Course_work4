Данная программа:
1. Получает информацию о вакансиях с платформ HeadHunter и SuperJob.
2. Сохраняет полученную информацию в файл
3. Позволять выводить информацию в консоль в следующем виде сортировки:
            1 - Вывести список вакансий (как есть)
            2 - Сортировать вакансии по минимальной зарплате (убывание)
            3 - Сортировать вакансии по минимальной зарплате (возрастание)
            4 - Сортировать вакансии по максимальной зарплате (возрастание)
